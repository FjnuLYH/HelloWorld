本项目主要分为3个文件：MainActivity、main.xml、list_item_arrays.xml


MainActivity作为程序入口，加载PreferenceScreen布局文件R.xml.main

main.xml包括了主要页面布局；以及直接嵌套PreferenceScreen跳转至其他页面；设置打开的网页目标

list_item_arrays.xml作为ListPreference的选项列表和各个选项的值的资源文件
